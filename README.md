🦖 FDINO TRANSPILER USER GUIDE (V1.0)

Welcome to FDino - An English-based programming language built with Rust for those who love minimalism and high performance!

🛠 1. INSTALLATION (For Beginners)
- Run the "Setup_FDino_V1.exe" file you just downloaded.
- Click "Next" and choose the installation folder.
- Check the box "Create a desktop shortcut" to show the icon on your screen.
- Click "Install" and wait a few seconds to complete.
- Click "Finish" to complete and exit setup

🚀 2. HOW TO RUN THE PROGRAM (Important!)
After installation, you will see the files in the installation folder. To translate your code, follow these steps:

Step 1: Open the "main.fdino" file with Notepad or VS Code. This is where you write your English code (e.g., the logic commands supported by FDino).
Step 2: Save the "main.fdino" file.
Step 3: Double-click the "main.exe" file (or the FDino icon on your Desktop).
Mechanism: "main.exe" is the brain (Transpiler). When you run it, it automatically finds the "main.fdino" file, reads your English code, and translates it into machine-executable logic.
Step 4: Check the results displayed on the console screen.

📂 3. INCLUDED FILES
- main.exe: The core Transpiler, written in Rust.
- main.fdino: A sample source file for you to get started.
- main.rs: The original source code of the transpiler (for those who want to study how FDino works).
